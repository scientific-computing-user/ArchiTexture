# Release Notes

This public release follows the final paper framing:

- central question: **does frozen SAM understand texture?**
- two paper-facing evidence routes:
  - **feature-space exploration**
  - **proposal-/mask-space exploration**
- historical anchors: **RWTD** and **STLD**
- broader main-story complements: **ADE20K-selected** and **ControlNet bridge**
- appendix real-world complement: **CAID**

Out-of-scope material from earlier internal branches is not part of this release's main narrative. In particular, Detector-style legacy branches and AdaSam-style adaptor results are not promoted here.
