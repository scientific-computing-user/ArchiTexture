# Reproducibility

This directory provides two entry points:

- [REPRODUCE_MAIN_RESULTS.md](REPRODUCE_MAIN_RESULTS.md): fastest path for the main-paper tables, matched comparisons, and paper build
- [REPRODUCE_APPENDIX.md](REPRODUCE_APPENDIX.md): appendix figures, descriptor ablation, and supporting-route assets

The repo ships the retained summary artifacts used in the paper. Full end-to-end reruns still require local benchmark roots, SAM upstream assets, and pretrained checkpoints described in `../checkpoints_manifest/`.
